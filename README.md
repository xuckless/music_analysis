# # Music Analysis

A Python project for analyzing and comparing music or audio pitch.

## Setup

### 1. Clone the repository:
   ```bash
   git clone https://github.com/xuckless/music_analysis.git
   cd music_analysis
   ```

### 2. Install Depenedencies
  ```bash
  pip3 install -r requirements.txt
  ```

### 3. Install FFMPEG
```bash
brew install ffmpeg
```

### 4. Run Main.py
```bash
python3 main.py
```
